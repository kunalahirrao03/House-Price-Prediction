# House-Price-Prediction
🏠 House Price Prediction 

📌 Overview
This project implements a House Price Prediction system using Multiple Linear Regression trained via Gradient Descent. The model predicts house prices based on features like area, number of bedrooms, and bathrooms.
Unlike basic implementations, this project builds the regression model from scratch, including:

 Cost Function (Mean Squared Error)
 
 Gradient Descent optimization
 
 Parameter tuning
